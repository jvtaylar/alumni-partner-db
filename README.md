# Alumni Partner DB

## Project Documentation
This project serves as a database for alumni partnerships, where organizations can connect with alumni for various opportunities. 

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/jvtaylar/alumni-partner-db.git
   ```

2. **Install dependencies**:
   Navigate to the project directory and run:
   ```bash
   npm install
   ```

3. **Run the application**:
   Start the application using:
   ```bash
   npm start
   ```

4. **Access the application**:
   Open a web browser and go to `http://localhost:3000` to view the application.
